Imagens No HTML

Resumo: As imagens são ferramentas que facilitam a compreensão de um assunto. Aprender a integrar imagens em sua página é um passo essencial para desenvolver sites cada vez mais interessantes. Existem diversos tipos de imagens e várias extensões que podemos utilizar, como por exemplo: (jpg, png, gif...).

Tags Atributos Função
<img> Src (Source) Liga um arquivo a nossa pagína
<img> Alt Nomeia a imagem (Obrigatorio)
<img> Width e Height Dimencionam uma imagem nos eixos (x) e (y).

Dimencionamento: width e Height

Quando quemos modificar o tamanho de uma imagem, recorremos a dois atributos Width (largura) e Height (altura), estes atributos possibilitam que você modifique o tamanho da imagem com unidades de medidas simples (pixels e porcentagens), observe a baixo uma imagem ilustrativa.


Pixeis:

quando queremos fazer algo para um tipo de tela específica, utilizamos pixeis, pois é uma medida padrão, e não responsiva por padrão.

leitura de imagem: Quando utilizamos pixeis para dimencionar uma imagem, está não se redimenciona conforme a tela, porém, cria-se uma barra de rolagem para que você possa visualizar-la utilizandoa

Porcentagem:

Porcentagem é variável dependente da tela em que é exibida, então, caso seja redimensionada, ela se modificará para exibir a porcentagem de tela devida.

leitura de imagem: está se dimenciona conforme o tamanho da tela de exibição, invez de criar uma barra de rolagem, está se dimenciona conforme o tamanho da tela, porcentagem é um tipo de medida responsiva.

Aplicando os valores

Os atributos width e height podem ser preenchidos com valores expecificos, a contagem de pixel vai varear do tamanho em que você deseja aplicar em uma imagem, estes tipos de atributos são utilizados no css inline ou externo.
