Tipos de Dados em JavaScript.


Em JavaScript, existem diversos tipos de dados que o programa pode manipular. Esses dados são classificados como primitivos, pois são básicos e de fácil compreensão. Vamos explorá-los:


String: É um tipo de dado formado por caracteres, podendo ser qualquer caractere. Para indicar que algo é uma string, usamos aspas simples ou duplas.

Number: Representa valores numéricos e é essencial para cálculos aritméticos e operações matemáticas em JavaScript.

Boolean: São dados simples que indicam se algo é verdadeiro (true) ou falso (false). São úteis para controle de fluxo e lógica em programas.



Tipo Número.


Os números são essenciais em diversas áreas, como matemática, física e química. Em JavaScript, podemos usá-los para realizar cálculos, medições de tempo e obter médias.

Ex:

var num1 = 1; //Declarando uma variável


var num2 = 2; //Declarando uma variável


var soma = num1 + num2; // Somando duas variáveis. Resultado: 3


Número Flutuante

Em JavaScript, números com casas decimais são representados usando pontos em vez de vírgula.

Ex:

var numero1 = 2.5; // Variavel com valor de numero flutuante


var numero2 = 10 / 3; // Resultado: 3.333333


Tipo String.

As strings contêm caracteres, como letras, números e símbolos especiais. Diferentemente dos números, as strings não podem ser usadas em operações aritméticas, podemos apenas concatenar as variáveis; (concantenação = juntar em sêquencia)

Ex:

Var nome = "João"; // Variavel com valor String


Var idade = '17' // Variavel com valor String


Var nickname = "Bolsonminion17"; // Variavel com valor String


Área Head

o head comporta as definições da nossa página, link entre um arquivo ou outro, qual idioma que está escrito o conteúdo, titulo da página entre outros.


Principais itens presentes no head:

<head>: Abre a seção de cabeçalho do documento html, onde são colocadas informações sobre páginas.

<meta charset=”utf-8”>: define o conjunto de caracteres como UTF-8, garantindo que o navegador interprete corretamente os caracteres especiais.

<meta name=”viewport” content=”width=DEVICE_Width, initial-scale=1.0”>: define a largura do tela e a escala inicial para dispositivos móveis, apresentando que a página seja exibida corretamente em diferentes tamanhos de tela.

<title> Tela inicial </title>: Define o titulo da página que será exibido na barrra de título do navegador.

</head>: Fecha a seção do cabeçalho.



Não é nessesario lidarmos com está parte ainda, se quiser, pode lidar com isso por conta, mas por enquanto, vamos focar na nossa página o que realmente vemos quando estamos projetando um site.


Área Body

O body é a parte mais importante desta atividade, é apartir desta área que poderemos criar nosso primeiros paragrafos, titulos implementar nossas imagens e uma variedade de elementos que você verá aqui neste site.



Um exemplo simples de titulo e paragrafo:

<body>: Abre a seção de corpo do conteúdo html, onde o conteúdo visível da página é colocado.

<h1>:Cria um titulo na nossa pagína</h1>

<p>: Cria um paragrafo na nossa página</p>


Colocando na prática.
