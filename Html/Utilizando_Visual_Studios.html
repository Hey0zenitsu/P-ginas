Preparando a Área de Trabalho

Introdução: esse é o completo zero, você não teve muita experiencia com algum programa de edição de codigo e se vê na obrigação de aprender né? bem chato de primeira vista, ter q usar um programa diferente pra fazer algo, porém, nem tudo é tão dificil como parece, há otimas escolhas e aqui te mostrarei a mais vantajosa!

Apresentação: Este é o Visual studio code, um dos programas mais usados para programar, disponivel na maioria dos computadores de instituições de ensino, provavelmente vc já deve ter visto e não deu tanta atenção, porém, essa ferramenta, abre portas para lidarmos com as mais diversas linguagem de programação. alguns exemplos são o python, java e javaScript.


Colocando a mão na massa: vamos lá, espero que você saiba o básico de informatica para lidar com esse programa, caso não saiba, peça ajuda de alguém que tenha mais experiencia ok!. observamos na tela inicial varias coisas diferentes, mas podemos visualizar com um distaque maior a sesção start contendo alguns textos em baixo correto?

Passo 01: podemos ver varios botões, mas apenas os três primeiros são os que utilizaremos, quando apertarmos o primeiro deles,veremos uma area de texto se abrir, essa área deve ser preenchida com o nome do nosso arquivo.

Passo 02: após nomearmos o arquivo, pressinamos enter para confirmar o nome, depois disso, um guia se abrirá, este é o explorador de arquivos, com ele procuraremos algum lugar para salvarmos o nosso arquivo.

Passo 03: essa parte pode ser um pouquinho complicada pra quem não sabe lidar com pastas e arquivos diversos, mas vamos com calma, na parte esquerda do explorador de arquivos temos alguns botões, estes botões são locais do nosso computador, a area de trabalho é o que vemos quando ligamos o computador, há aplicativos diversos talvez pastas, há uma infinidade de coisas que podem ter aí no seu computador.

Passo 04: está é a minha area de trabalho, cheia de aplicativos, aqui temos pastas, programas, arquivos diversos. na area de trabalho, podemos salvar o nosso arquivo, porém, é recomendavel criar uma pasta para guardar seu arquivo, para manter seu computador mais organizado.

Passo 05: para criarmos a pasta, apertamos botão direito do mouse na area onde os aplicativos estão, depois disso, uma area abrirá, basta fazer igual na imagem e assim criaremos a pasta.

Passo 07: a nossa pasta foi criada, agora basta nomearmos ela.

Passo 08: Após isso, abrimos a nossa pasta, e podemos em fim salvar o nosso arquivo.

Final. podemos agora criar o nosso código.

agora temos completo controle sobre aquela pasta que criamos anteriormente. ainda na pasta, podemos observar a existencia de botões no interior da pasta, veja só.

Essa é uma função bem importante, estes botões servem para que possamos criar novas pastas ou arquivos no interior do nosso projeto, o primeiro cria um arquivo de extenção txt, já o segundo cria uma pasta dentro do projeto, agora que você sabe um pouco mais de arquivos e pastas, porque não falarmos sobre os tipos de arquivos?


Tipos de arquivos


Resumo: Nos dias de hoje, temos diversos tipos de linguagem de programação, diversos programas e varios tipos de arquivos, os arquivos são aglomerados de informações que são processados por um programa. um bom exemplo é as imagens que são um tipo de arquivo geralmente associado a (png, jpg, gif, webp), estes são alguns tipos de arquivos de imagem. O Vscode funciona como um editor de texto, podemos escrever o código, e por meio de algum programa este é executado, porém, para que a execução seja feita, devemos sempre expecificar qual é o tipo do nosso arquivo, afinal, há uma infinidade de tipos para se lidar.

Como modificar o tipo de arquivo?


Cada linguagem de programação tem um tipo de arquivo diferente, python por exemplo atende pelo tipo (.py), html atende pelo tipo (.html), nem sempre é previsivel qual a terminação a utilizar, mas, vamos te dar uma ajudinha, veja uma tabela de extenções a baixo.

Tipo de arquivo Linguagem Exemplo
.html Linguagem Html MeuArquivo.html
.css Linguagem CSS MeuArquivo.css
.js Linguagem JavaScript MeuArquivo.js
.java Linguagem java MeuArquivo.java
.py Linguagem Python Dados da tabela

como mudar o tipo de arquivo.

para modiicarmos um tipo de arquivo, basta apenas mdificar a terminação do arquivo, ou seja se o arquivo se chama (meuarquivo.txt) podemos mudar para (meuarquivo.html), assim tornamos o arquivo no tipo html.

Processo:

Passo 1: primeiro devemos apertar com botão direito no arquivo, e apertar em renomear, veja abaixo.

depois disso, basta mudar a terminação de txt para a terminação que quisermos, no meu caso, mudamos para uma terminação html.

finalização: depois que mudamos o tipo de arquivo, tudo o que quisermos escrever no campo de texto, sera identificado como uma linha de código.
