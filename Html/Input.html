Formulários em HTML

Resumo: Os formulários são elementos de registro, o papel dos <inputs> São adicionar uma área que pode ser respondida, seja com alternativas ou escrita, os inputs exercem um papel importante para uma página, pois é por meio destas que podemos fazer ferramentas utilizando JavaScript, áreas de registro e login para maior segurança, então aprender a utilizar os inputs são essenciais para quem quer dar um passo a mais no Front-end.

Tags atributos Função
<Form> Action Cria uma estrutura de formulario.
<Label> Style Cria um nome de identificação na página.
<Input> Type Acrescenta um item interativo (varia).
<Section> Style Cria uma estrutura espancivel, contendo varias outras alternativas.
<Textarea> Style acrescenta uma area de texto, preenchivel com qualquer caracter.

Como criar um Input

A tag input é uma tag de entrada, e existem variações, para criarmos tipos diferentes de input utilizamos o atributo type (tipo) para modificar o tipo do item.

<Form>: para criar um formulario, utilizamos a tag <Form> e acrescentamos os inputs em seu meio com a tag <input>. depois definimos de que tipo esse input é utilizando o atributo type.

Os botões são itens clicáveis que podem ter suas funcionalidades modificadas. Existem dois tipos: o tipo button e o tipo submit. O botão normal não faz nada sem ter uma programação prévia. O botão submit tem a função de enviar o que está dentro do formulário para algum destino, seja um banco de dados ou outro tipo de arquivo de processamento.
