Introdução

Para está atividade utilizaremos o terminal do VsCode, segue a imagem de como abrir. aperte em "new terminal".

o terminal é uma ferramenta que utilizamos para diversas funções, podemos criar arquivos e pastas, podemos executar um programa, extrair ou compactar arquivos, há uma infinidade de funções, mas hoje, vou utilizar um comando que cria um arquivo, chamado "code".

após termos aberto um novo arquivo, podemos enfim codar em python. para iniciarmos, vamos usar o mais velho exemplo de python, print ("hello, world").

Vamos explicar o que está acontecendo. o comando print, estampa uma mensagem no terminal, quando escrevemos print("conteúdo a ser estampado") e apertamos no botão de execução de código, vemos que no terminal há uma mensagem, hello, world, a mesma mensagem que está especificada na linha de código.


Lidando com erros
