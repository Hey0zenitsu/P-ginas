Listas No HTML

Resumo: As listas desempenham um papel essencial na organização hierárquica ou sequencial de informações, proporcionando uma compreensão
mais clara e estruturada de dados relacionados a um determinado tema.

Tags Tradução Função
<ol> Ordered Lists Criar uma Lista Ordenada.
<ul> Unordered Lists Criar uma Lista não Ordenada.
<li> List information Acrescenta Uma Informação a Alguma Lista.

Listas Não Ordenadas

As listas não ordenadas são listas que não contem uma ordem ou hierarquia previa, exibindo informações com um caracter simples.

<ul>: para criar uma lista sem ordem primeiro colocamos a tag <ul> e acrescentamos informações em seu meio com a tag <li>.

Listas Ordenadas

As listas ordenadas são listas que contem uma ordem ou hierarquia previa, exibindo informações com uma ordem.

<ol>: para criar uma lista com ordem primeiro colocamos a tag <ol> e acrescentamos informações em seu meio com a tag <li>.

Resultado

  1. Html
  2. CSS
  3. JavaScript
  4. TypeScript


Podemos acrescentar o atributo type (tipo) na tabela com ordem, este atributo muda a forma que ordenamos os itens da nossa lista, temos quatro tipos de ordens que podemos utilizar na nossa lista sendo eles o a, A, I e 1 (Letras minusculas, Letras Maiusculas, Algarismos romanos, Numeros)
